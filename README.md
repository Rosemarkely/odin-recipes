# odin-recipes

This project features the creation of a recipe site in which is shown a few recipes along with images and what you need and for the recipes along with how to make them

In this project I demonstrated the ability to create a basic html website that includes links, images along with text for readability as well as some support for the disabled as images come with descriptions. This was a fun project and I cannot wait until I am able to fully create a fully running website with all the intended mechanisms and more.
